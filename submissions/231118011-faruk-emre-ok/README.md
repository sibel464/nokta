## 📦 Kurulum ve Çalıştırma
Proje Expo (React Native) kullanılarak geliştirilmiştir.

1. `app` klasörüne gidin: `cd app`
2. Bağımlılıkları yükleyin: `npm install`
3. Uygulamayı başlatın: `npx expo start`

*Not: Uygulamanın çalışması için `index.tsx` ve `spec.tsx` dosyalarına geçerli bir Google Gemini API Key eklenmelidir.*
